# online_project

王一互動資訊-模擬專案

## 啟動方式

```bash
npm install
npm run dev
```

## 技術選用

- Vue 3 + Nuxt 3
- 純 CSS（無 UI 框架）
- Google Fonts — Roboto

## 專案結構

```
pages/
  index.vue          # 主頁面，整體佈局與資料
components/
  AppNav.vue         # 頂部導航列
  ProductViewer.vue  # 左側圖片區塊與模式切換
  ProductInfo.vue    # 右側產品資訊
assets/css/
  main.css           # 全域樣式與 CSS 變數
public/images/       # 圖片資源
```
